Spotify API Web Scraping Project



Purpose: Provide more interesting data to help Spotify users find songs they will like based on specific criteria/data from Spotify's API


Project Requirements:
  • Use Spotify API to pull data on Spotify tracks audio features
  • Analyze and vizualize data points over time
  • Allow users to view, filter, and search for songs that match specific features


Data will be taken from Spotify API at SpotifyforDevelopers

Data points:
Get tracks audio analysis: would be interesting to follow/analyze and also plot against each other in a graph
  • accousticness
  • danceability
  • duration_ms
  • enery
  • instrumentalness
  • loudness
  • valence
Get track: basic data I should include:
  • genres
  • name
  • popularity
  • duration
  • explicit bool


Examples:
  • A user wants a song with danceabillity of 0.8 and energy > 0.7. The search feature will pull up a list of Spotify links for songs that fit that criteria.
  • A user wants to look up how a track's popularity changed over time. The graph feature will pull up a graph showing this data.
